서비스 소개 (About Us)

'관상노리'는 현대적인 인공지능 기술과 전통적인 관상학적 특징을 결합하여 사용자에게 재미와 통찰을 제공하는 엔터테인먼트 서비스입니다. Google의 Teachable Machine을 활용하여 수만 장의 유명인 데이터를 학습시킨 모델을 통해, 당신의 얼굴이 가진 고유한 특징이 어떤 '상'에 가까운지 분석해 드립니다.

우리는 단순히 결과를 보여주는 것에 그치지 않고, 각 관상이 가진 긍정적인 의미와 특징을 상세히 설명하여 사용자들이 자신의 매력을 재발견할 수 있도록 돕습니다.

자주 묻는 질문 (FAQ)

Q: 내 사진이 서버에 저장되나요?

A: 아니요. '관상노리'는 프라이버시를 최우선으로 생각합니다. 모든 이미지 분석은 사용자의 브라우저 내에서 직접 수행되며, 어떤 이미지도 서버로 전송되거나 저장되지 않습니다.

Q: 분석 결과가 정확한가요?

A: 본 서비스는 엔터테인먼트 목적으로 제작되었습니다. AI 모델은 수만 장의 유명인 데이터를 학습하여 통계적인 유사성을 찾을 뿐, 실제 관상학적 운명을 보장하지는 않습니다. 재미로 즐겨주세요!

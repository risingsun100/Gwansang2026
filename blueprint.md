# 관상노리 (Gwansangnori) Blueprint

## 프로젝트 개요
'관상노리'는 Google의 Teachable Machine 기술을 활용하여 사용자의 얼굴 사진을 분석하고, 그와 닮은 관상(가수상, 영화배우상, 연예인상, 정치법률가상, 부자상 등)을 찾아주는 인공지능 테스트 웹 서비스입니다.

## 주요 기능
- **성별 선택:** 사용자가 성별을 선택하여 최적화된 AI 모델(남성/여성용)을 로드할 수 있습니다.
- **이미지 업로드 및 분석:** 드래그 앤 드롭 또는 파일 선택을 통해 얼굴 이미지를 업로드하면 실시간으로 분석이 진행됩니다.
- **결과 표시:** AI 분석 결과를 바탕으로 관상의 종류, 설명, 그리고 닮은 유명인(셀럽) 목록을 보여줍니다.
- **확률 차트:** 각 관상별 일치 확률을 시각적인 막대 그래프로 표시합니다.
- **재시도 및 공유:** 다른 사진으로 다시 테스트하거나 결과를 공유할 수 있는 기능을 제공합니다.

## 디자인 및 스타일 가이드 (현대적 웹 표준 적용)
- **컬러 시스템:** `oklch` 색상 공간을 사용하여 생동감 넘치는 색상을 적용합니다. CSS 변수를 통해 테마를 관리합니다.
- **레이아웃:** 컨테이너 쿼리(`@container`)를 사용하여 부모 요소 크기에 반응하는 컴포넌트를 설계합니다.
- **시각적 효과:** 다층 그림자(Multi-layered shadows)를 사용하여 깊이감을 부여하고, 배경에는 미세한 노이즈 질감을 추가하여 프리미엄 느낌을 줍니다.
- **인터랙티브 요소:** 버튼과 토글에 글로우(Glow) 효과와 부드러운 전환(Transition)을 적용합니다.
- **타이포그래피:** 'Jua' 폰트를 메인으로 사용하며, 시각적 계층 구조를 명확히 하기 위해 폰트 크기와 두께를 조절합니다.

## 구현 계획 (단계별)

### 1단계: 코드 구조 개선
- `index.html`에 포함된 자바스크립트 로직을 `main.js`로 분리 및 모듈화.
- 불필요한 인라인 스타일 제거 및 구조 정리.

### 2단계: 스타일링 (CSS 현대화)
- `style.css`에 `oklch`, CSS 변수, `@layer` 등을 적용하여 전면적인 디자인 개편.
- 반응형 웹 디자인 강화 (모바일 우선 접근).
- 로딩 애니메이션 및 결과 표시 영역의 시각적 완성도 향상.

### 3단계: 접근성 및 UX 강화
- 스크린 리더 지원을 위한 ARIA 속성 보완.
- 키보드 내비게이션 지원.
- 이미지 업로드 시 피드백 강화.

### 4단계: 기능 보완
- 현재 모델의 결과를 좀 더 풍부하게 보여줄 수 있는 UI 요소 추가.
- 에러 핸들링 (이미지 분석 실패 등) 추가.
